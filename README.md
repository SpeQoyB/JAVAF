# University Management System

Невелика консольна Java‑система для керування студентами, викладачами, курсами та зарахуваннями.

## Структура проєкту

- entities/ — сутності (Student, Teacher, Course, Enrollment)
- enums/ — переліки (StudentStatus, TeacherPosition, Grade)
- services/ — логіка керування (StudentService, TeacherService, CourseService, EnrollmentService)
- util/ — утилітарні методи (GPAUtils)
- Main.java — консольне меню та взаємодія з користувачем
- 
### Студенти

- додавання нового студента
- оновлення даних
- зміна статусу (ACTIVE, ON_LEAVE, EXPELLED, GRADUATED)
- видалення
- фільтрація та сортування
- пошук за ПІБ або email

### Викладачі

- додавання
- оновлення
- видалення
- перегляд списку

### Курси

- створення курсу
- прив’язка викладача
- оновлення
- видалення
- перегляд усіх курсів

### Зарахування

- зарахування студента на курс
- виставлення оцінки
- позначення оплати
- перегляд транскрипту
- обчислення GPA студента

### Звіти

- пошук студентів
- список боржників
- середній GPA по курсу

## Технології

- Java 17+
- Консольний інтерфейс
- ООП: наслідування, енумерації, інтерфейси
- Робота з масивами без колекцій (навчальний формат)

## Як запустити

1. Клонуйте репозиторій:
   git clone <url>
2. Відкрийте проєкт у будь‑якому IDE (IntelliJ IDEA, Eclipse, VS Code)
3. Запустіть файл Main.java

## Обчислення GPA

GPA рахується на основі оцінок студента:
- A = 4.0
- B = 3.0
- C = 2.0
- D = 1.0
- F = 0.0
- NA не враховується
